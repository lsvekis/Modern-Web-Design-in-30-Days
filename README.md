# Modern Web Design in 30 Days — Source Code (GitHub Repo)

This repository contains **starter files + completed examples** used throughout the book
**Modern Web Design in 30 Days** by **Laurence Lars Svekis**.

## Quick Start
- Open any `dayXX/index.html` in your browser.
- Open DevTools → Console to see logs and errors.
- Edit files and refresh to experiment.

## Structure
- `day01` … `day30` — daily folders
- Each folder includes:
  - `index.html`
  - `styles.css`
  - `script.js`
  - `notes.md`

## Notes
Days 21–23 include a richer, complete mini-project (“Dynamic Personal Website”).
Day 24 includes a debugging playground.
